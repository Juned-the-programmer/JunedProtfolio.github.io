God's Eye

I have took the inspiration from the fast and furiouos 7 movie in that they have introduce the God's Eye , Which can track any person in the world from anywhere.

I have tried to make a similar kind of thing but not in the that much big architecture. Just a small version of that we can track any person in the collage or a one village easily.

The person data are stored in one centralized database. I have used the dlib library to extract the landmarks of the face and and store in the database.

I have used 2 component in this (1) Frontend (2) Backend. We have to run the server (Backend) on each camera service , And it will compare it will recognize using the facial recognition library.

I have used the socketIO in that so if any person got recognized in any camera which we are finding. It will directly send the request to the frontend to display it. That the person has appered at this camera at this time.

Technologies:
  • - Python
  • - Flask
  • - Open CV
  • - Face Recognition
  • - Python-Engine-IO
  • - Python-socket-IO
