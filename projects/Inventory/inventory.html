Inventory Management System

This was a project I started in back 2020 when i have good understanding of Djago Framework . In this Project i have included each and every functionality which is offered by customer.

This system manage the Inventory of Particular shop with Billing and customer management also .

From this system the the owner can manage almost everything they want. Some other companies offers this kind of applications but they provide just a Stock management.

They have all the data and it is hosted on web server. So they can easily check the details where they want to check.

I worked on this projects almost 6 months to Completely Workabl web application.



This application is for only Estimate Sale and Purchase.Currently I have created a GST application but i gave it for testing to the client.
Technologies:
  • - Python
  • - Django
  • - MySQL
  • - AWS (S3 & RDS)
  • - Heroku(For Hosting)
  • - Jquery
